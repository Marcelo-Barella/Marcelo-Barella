## Cursor AI & agentic tooling

### [cursor-sync](https://github.com/Marcelo-Barella/cursor-sync)
Cursor extension that syncs user-level settings, rules, skills, and keybindings to a private GitHub Gist. Supports manual push/pull, scheduled sync, conflict resolution, export/import via public Gists, and configurable extension sync.

### [Cursor-CA-Orchestrator](https://github.com/Marcelo-Barella/Cursor-CA-Orchestrator)
TypeScript CLI that orchestrates multiple Cursor Cloud Agents working across different GitHub repositories. Provisions a bootstrap repo, uses per-run branches as a coordination bulletin board, and launches an orchestrator agent that manages worker agents targeting your repos. Includes an interactive REPL with slash commands, single-prompt mode with automatic task decomposition, and CI/non-interactive mode.

### [Agentic-Browser-AI](https://github.com/Marcelo-Barella/Agentic-Browser-AI)
MCP server implementation (InDOM) with SSE integration for Cursor IDE. Provides advanced DOM inspection, intelligent screenshot capture with semantic analysis, browser automation, a comprehensive testing framework, and project analysis tools.

### [AI_SDK-Prototype](https://github.com/Marcelo-Barella/AI_SDK-Prototype)
AI-powered Vue 3 component enhancement system using Vercel AI SDK v5 and Model Context Protocol. Analyzes Vue SFCs and generates enhancement variations for performance, accessibility, and quality with visual previews via Puppeteer and persistent memory via SQLite.
